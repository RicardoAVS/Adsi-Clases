MATERIAL CSS

Las Hojas de Estilo en Cascada (CSS) es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento escrito en HTML o XML (incluidos los dialectos XML como SVG, MathML o XHTML). Las CSS describen cómo deben presentarse los elementos en la pantalla, en el papel, en el habla o en otros medios.

El CSS es uno de los lenguajes centrales de la Web abierta y está estandarizado en todos los navegadores Web de acuerdo con la especificación del W3C. Desarrollado en niveles, CSS1 es ahora obsoleto, CSS2.1 es una recomendación, y CSS3, ahora dividido en módulos más pequeños, está progresando en la vía de la estandarización.
