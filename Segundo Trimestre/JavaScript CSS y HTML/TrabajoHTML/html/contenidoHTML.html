Material sobre HTML

HTML (HyperText Markup Language) es el bloque de construcción más básico de la Web. Define el significado y la estructura del contenido de la web. Otras tecnologías además del HTML se utilizan generalmente para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).

"Hipertexto" se refiere a los enlaces que conectan las páginas web entre sí, ya sea dentro de un mismo sitio web o entre sitios web. Los enlaces son un aspecto fundamental de la Web. Al cargar contenido en Internet y enlazarlo con páginas creadas por otras personas, uno se convierte en un participante activo de la World Wide Web.

El HTML utiliza el "marcado" para anotar texto, imágenes y otros contenidos para su visualización en un navegador de la Web. El marcado HTML incluye "elementos" especiales como <title>, <head>, <body>, <header>, <footer>, <article>, <section>, <p>, <div>, <span>, <img>, <aside>, <audio>, <canvas>, <datalist>, <details>, <embed>, <nav>, <output>, <progress>, <video>, <ul>, <ol>, <li>, and many others.

Un elemento HTML se separa de otro texto en un documento por "tags", que consisten en el nombre del elemento rodeado por "<" y ">" . El nombre de un elemento dentro de una etiqueta no distingue entre mayúsculas y minúsculas. Es decir, puede ser escrito en mayúsculas, minúsculas o una mezcla. Por ejemplo, la etiqueta <title> puede escribirse como <Title>, <TITLE>, o de cualquier otra manera.
