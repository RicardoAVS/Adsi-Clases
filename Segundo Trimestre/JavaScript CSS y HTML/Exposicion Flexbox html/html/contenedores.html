Propiedades de Contenedores

  1. Propiedad: Direction
  2. Propiedad: Wrap
  3. Propiedad: Flow
  4. Propiedad: Justify
  5. Propiedad: Align

1. Propiedad: Direction

flex-direction

Esta propiedad recibe los valores de: row,column,row-reverse y column-reverse.

2. Propiedad: wrap

flex-wrap

Esta propiedad recibe los valores de: nowrap,wrap y wrap-reverse.

3. Propiedad: Direction and wrap

flex-flow

Esta propiedad es una abreviación para usar las dos propiedades anteriores a la vez. Recibe primero el valor de flex-diretion y luego el valor de flex-wrap. Por ejemplo:
flex-flow: row wrap;

4. Propiedad: Justify

justify-content

Esta propiedad recibe los valores de: flex-start, flex-end, center, space-around y espace-between.
Sirve para alinear los elementos en su eje principal, es decir, si mi flex-direction es row o row-reverse mis elementos se van a acomodar en el eje horizontal, si por el contrario mi flex-direction es column o column-reverse entonces mis elementos se van a acomodar en el eje vertical.

5. Propiedad: Align

Esta propiedad recibe los valores de: flex-start, flex-end, center, stretch y baseline.
Sirve para alinear los elementos en el eje perpendicular a su eje principal, es decir, si mi flex-direction es row o row-reverse mis elementos se van a acomodar en el eje vertical, si por el contrario mi flex-direction es column o column-reverse entonces mis elementos se van a acomodar en el eje horizontal.

align-items

Esta propiedad se usa para grupos de elementos de un fila, permite alinear todos los elementos del contenedor, incluidos los elementos independientes.

align-content

Esta propiedad se usa para grupos de elementos de dos o más filas, permite alinear las líneas/filas de elementos de un contenedor.
