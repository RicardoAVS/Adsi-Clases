Propiedades de Elementos

  1. Propiedad: Flex
  2. Propiedad: Align-self
  3. Propiedad: Order

1. Propiedad: flex

flex-basis

Esta propiedad recibe el valor a partir del cuál un elemento va a crecer o decrecer cuando se necesario.

2. Propiedad: align

align-self

Esta propiedad se usa para elementos específicos, permite alinear elementos independientes del contenedor.
Si está ordenado en row, el elemento se alineará verticalmente.
Si está ordenado en column sel elemento se alineará horizontalmente.

3. Propiedad: order

order

La propiedad order de CSS establece el orden para un item en un contenedor flex o grid.Los items en un contenedor estan ordenados de forma ascendente y luego por su orden del codigo fuente.
